웹사이트를 실행하려면 터미널(PowerShell)에 npm run dev를 입력하면 됩니다.

만약 이전에 발생했던 **권한 오류(Execution Policy)**가 다시 나온다면, 아래 명령어를 복사해서 입력해 주세요. 이 명령어는 현재 세션에서만 스크립트 실행 권한을 일시적으로 허용하고 개발 서버를 시작합니다.

Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass; npm run dev
